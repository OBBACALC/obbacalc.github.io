What Each Status Means

✅ No State Income Tax

State has no income tax — only the federal OBBBA deduction applies, which you receive in full.

✅ Conforms to OBBBA

State automatically adopts federal changes. You receive the deduction at both federal and state level.

🔍 Decision Pending

Static conformity state — must pass legislation to adopt OBBBA. Decision expected before April 15, 2026.

⚠️ Decoupled

State has explicitly chosen not to conform. Your federal deduction is unaffected — no state-level benefit.
